Botón píldora de Pump&Flow para toda acción; primario lavanda, siempre redondeado.

```jsx
<Button variant="primary" size="lg">Reserva tu lugar ✨</Button>
<Button variant="secondary">Ver horarios</Button>
```

Variantes: `primary` (lavanda, la acción principal), `secondary` (contorno lavanda), `dark` (morado), `onDark` (blanco, para bloques lavanda/morado), `ghost`. Tamaños sm/md/lg; md mínimo 44px de alto (target táctil).
